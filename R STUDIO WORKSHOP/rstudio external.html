Centre for Health Informatics Presents:
2-Day R Studio Workshop Event
This in-person workshop runs over 2 days and goes over the foundations of R & R Studio, as well as, working with data & data structures, creating data visualization and employing machine learning.
Day 1 (Introduction):
  • Getting started with R and R studio
  • Installing R
  • Getting familiar with GUI
  • Basic coding operations
  • Arithmetic operators
  • Code chunks
  • Storing/Using Variables
  • Logical Operators
  • Data types
  • Data Structures
  • Different types of datastuctures (Vectors, Lists, Matrices, Dataframes, Factors)
  • Data Wrangling
  • Loading and using packages
  • Loading datasets
  • Cleaning and combining a data set
Day 2 (Data-Exploration):
  • Summary statistics
  • Data visualization
  • Plots using base R
  • Plots with ggplot2
  • Summary table with gtsummary
  • Regression analysis
  • Other capabilities of R
  • Machine learning
  • R-bookdown
  • R-Shiny

11 p.m. - 2 p.m. MST
