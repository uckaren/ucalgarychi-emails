Centre for Health Informatics Presents:
Introduction to R Programming Workshop
Do you want to improve your R Studio skills, but struggle with a traditional online learning? Our In-Person 2 Day Introduction to R Programming Workshop is for you!
This workshop will introduce basic programming concepts in R and learning to execute data manipulations, calculations, basic statistical analyses, and produce useful figures and tables. Participants will also learn to write simple functions that can be used to automate analyses, practical statistical computing, and general programming concepts.
By the end of the workshop, participants will be able to install and configure R and R studio, be familiar with the R studio IDE, clean and prepare a dataset for analysis with common packages and functions, manipulate a data set to create meaningful tables and figures, perform some common statistical analysis and, learn about some other advanced capabilities that R has to offer.
Day 1 (Introduction):
  • Getting started with R and R studio
  • Installing R
  • Getting familiar with GUI
  • Basic coding operations
  • Arithmetic operators
  • Code chunks
  • Storing/Using Variables
  • Logical Operators
  • Data types
  • Data Structures
  • Different types of datastuctures (Vectors, Lists, Matrices, Dataframes, Factors)
  • Data Wrangling
  • Loading and using packages
  • Loading datasets
  • Cleaning and combining a data set
Day 2 (Data-Exploration):
  • R-bookdown
  • R-shiny
  • R-markdown
  • Machine Learning
